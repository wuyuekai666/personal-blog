# WuYuekai's personal Blog

Welcome to my personal course website.

This website is built with **Jekyll** and deployed using **GitHub Pages**. It is used to organize my course assignments, reports, source code, and documentation.

## About This Website

In this project, I practiced:

- creating a static personal website
- writing documentation with Markdown
- using Git for version control
- deploying a website with GitHub Pages
- integrating previous assignment work into one website
