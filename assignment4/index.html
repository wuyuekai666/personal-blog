2D eVTOL Site Selection and Path Planning Visualization System

A Streamlit-based research application for urban low-altitude mobility planning. It combines demand generation, K-means candidate site creation, genetic-algorithm site selection, and A* / RRT* path planning comparison.

Read Report Download Source ZIP Download Presentation

Module 1

Landing Site Selection

Generates passenger demand points, creates risk and cost fields, clusters demand with K-means, and uses a genetic algorithm to select final eVTOL landing sites.

Module 2

Path Planning

Reuses the selected sites as starting points and compares A* and RRT* routes to a fixed airport connection point with length, risk, cost, runtime, and tortuosity metrics.

Run Locally

Streamlit Application

pip install -r requirements.txt
streamlit run main.py
